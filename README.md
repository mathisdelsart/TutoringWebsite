## Stack

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** with custom animations
- **Lucide React** icons
- FR / EN bilingual UI
- Static export → **GitHub Pages**

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000/mathis-delsart](http://localhost:3000/mathis-delsart).
